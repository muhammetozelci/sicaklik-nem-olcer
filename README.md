#  Akıllı Sıcaklık ve Nem Ölçer

Bu projede DHT22 sensörü kullanılarak ortamın sıcaklık ve nem değerleri ölçülmekte ve LCD ekranda gösterilmektedir.

##  Kullanılanlar
- Arduino UNO
- DHT22 sensör
- LCD ekran

##  Özellikler
- Gerçek zamanlı ölçüm
- LCD ekranda gösterim
